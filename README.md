# t-wada-stand

> テスト書いてないとかお前それ@t-wadaの前で同じこと言えんの

テストを書いてない書いてない人の前に例のスタンドが現れます。

## Installation

```
$ git clone git@github.com:akameco/twada-stand.git
```

## Usage

```
$ npn install
$ npm start
```

デスクトップの前で、「テスト書いてない」と声に出してつぶやく。
もしくは、OSXのユーザーの場合、`say`コマンドを実行できます。

```
$ say テスト書いてない
```
